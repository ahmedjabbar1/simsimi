Installation

sudo apt-get update
-
sudo apt-get install libreadline-dev libconfig-dev libssl-dev lua5.2 liblua5.2-dev lua-socket lua-sec lua-expat libevent-dev make unzip git redis-server autoconf g++ libjansson-dev libpython-dev expat libexpat1-dev
-
git clone https://github.com/wathiq-iq/simsimi
-
cd simsimi
-
sh launch.sh

Setup the bot

  • open to config.lua to edit
  • edit the token bot bot_api_key='hear'
  • edit the token Api simsimi simsimi_key
  • edit the about bot about

chat with bot

Feel free to try the bot via Telegram or manually search @LAZUBOT in Telegram app.
